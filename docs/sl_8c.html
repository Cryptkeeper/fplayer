int sladd ( slist_t ** sl,
const char * str )

sladd() duplicates a string and appends it to a string list. The string list is dynamically resized as needed and returned via the sl param. Caller is responsible for freeing the list using slfree.

Parameters
slThe string list to append to, and the return value of the resized list.
strThe string to duplicate and append.
Returns
0 if successful and sl is set to the new string list, otherwise -1 is returned and errno is set.
